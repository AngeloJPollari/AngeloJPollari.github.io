Project Overview

This project simulates a small enterprise network with multiple security layers:

  • Internal LAN with VLAN segmentation
  • Core routing between main and remote sites
  • Cisco ASA firewall separating inside, DMZ, and outside zones
  • Central services including DHCP, NTP, and syslog
  • AAA authentication via TACACS+ and RADIUS
  • Wireless LAN Controller with employee and guest WLANs

Architecture & Security Components

4.1 Core LAN & VLAN Segmentation

Tools Used: Cisco 3650 Switches, Cisco 4331 Router

  • Layer 2 switching between core switches
  • VLAN 1 management interfaces with static IPs
  • Default gateways assigned per subnet
  • Connectivity verified with ICMP and ARP inspection

4.2 AAA, TACACS+, RADIUS

Tools Used: AAA Server, Cisco 4331 Routers

Configuration:

  • TACACS+ on router R1 and one switch
  • RADIUS on second switch
  • AAA new-model with fallback local authentication

4.3 IPsec VPN

Tools Used: Cisco 4331 Routers

  • ISAKMP Phase 1: AES-256, PSK, DH Group 5
  • Phase 2: ESP-AES-256, ESP-SHA-HMAC
  • Crypto maps applied to transit interfaces

4.4 ASA Firewall & DMZ

Tools Used: Cisco ASA 5506-X

  • Inside (100), DMZ (50), Outside (0)
  • NAT rules for DMZ and Inside networks
  • ACLs controlling HTTP/HTTPS and blocking FTP

4.5 Monitoring (Syslog, SNMP, IPS)

  • Syslog/NTP server for device logging
  • SNMP R/W communities tested via MIB Browser
  • IOS IPS applied on LAN-facing interface

4.6 DHCP & Address Management

  • DHCP scopes configured on ASA
  • WLC DHCP for wireless clients
  • Static IP diagrams updated after changes

4.7 Wireless LAN Controller

  • Staff WLAN – WPA2-PSK
  • Guest WLAN – WEP (legacy demo)
  • Roaming tested between APs

Implementation Timeline

Phase 1 – Core Switching & Routing

Configured IPs, VLAN1, default gateways, and verified connectivity.

Phase 2 – AAA with TACACS+/RADIUS

Centralized device authentication with fallback local credentials.

Phase 3 – IPsec VPN

Built encrypted tunnel between LANs using ISAKMP and crypto maps.

Phase 4 – ASA Firewall, DMZ, NTP, Syslog

Set up firewall zones, DHCP, NAT, ACLs, and centralized logging.

Phase 5 – SNMP & IPS

Enabled network monitoring and intrusion prevention signatures.

Phase 6 – Wireless LAN Segmentation

Integrated a WLC and lightweight access points, configured staff and guest SSIDs, verified DHCP assignments, and tested roaming and high availability.
